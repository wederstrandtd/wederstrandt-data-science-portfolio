# Statistics

## Scope
This folder contains statistical modeling coursework artifacts, including regression analysis and supporting data.

## Problem Context
This work compares modeling strategies for continuous or generalized outcomes, with an emphasis on selecting model forms that align with the data-generating assumptions.

## Contents
- `Comparing Regression and Generalized Linear Models.docx`: Written comparison of regression frameworks.
- `Housing.csv`: Dataset used in statistical exercises.
- `week_11_wederstrandt_d.Rmd`: R Markdown source for week 11 analysis.
- `week_11_wederstrandt_d.pdf`: Rendered analysis output.

## Technical Focus
- Classical regression and generalized linear model (GLM) comparison.
- Statistical inference and model interpretation.
- Reproducible analysis reporting with R Markdown.

## Tooling
- R / R Markdown
- CSV-based tabular data workflows

## Methodological Themes
- Model specification and assumption checking.
- Interpretation of coefficients, uncertainty, and effect direction.
- Fit diagnostics and comparative model reasoning.
- Communication of statistical evidence in report form.

## Reproducibility Steps
1. Open the `.Rmd` file in an R-enabled environment.
2. Ensure required R packages are installed.
3. Confirm dataset path references align with `Housing.csv` location.
4. Knit/render the report to reproduce analysis outputs.

## Expected Outputs
- A rendered analysis report (`.pdf`) documenting model setup, diagnostics, and interpretation.
- Structured narrative comparing regression and GLM suitability.

## Notes
- File set includes both source and rendered outputs for easier review.
- The folder emphasizes interpretability and inferential reasoning over production deployment.
