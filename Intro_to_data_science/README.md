# Intro_to_data_science

## Scope
This directory stores introductory data science project deliverables focused on an energy drink analysis.

## Project Goal
Establish a foundational analytics narrative by defining a business-relevant question, collecting supporting evidence, and translating findings into written and presentation formats.

## Contents
- `Energy Drink Paper.docx`: Written analysis report.
- `Energy Drink Presentation.pdf`: Exported presentation.
- `Energy Drink Presentation.pptx`: Slide deck source.

## Technical Focus
- Early-stage analytical framing and communication.
- Translation of data science findings into written and presentation formats.

## Analytical Components Covered
- Problem definition and hypothesis framing.
- Basic descriptive analysis and interpretation.
- Audience-focused result communication.

## Deliverable Roles
- Paper (`.docx`): Detailed methodology and findings narrative.
- Slides (`.pptx`, `.pdf`): Executive-level summary and visual communication.

## How To Use This Folder
1. Review the paper first for the full analytical context.
2. Use the presentation deck for condensed findings and key visuals.
3. Compare slide framing against report detail to understand communication trade-offs.

## Notes
These assets are primarily final deliverables; supporting code and raw data may reside outside this directory.

## Limitations
- Source notebooks and intermediate data processing steps are not stored in this folder.
- Reproducibility is documentation-first rather than code-first for this early-stage project.
